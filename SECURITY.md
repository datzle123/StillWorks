# Security Policy

MergeVow will execute candidate application code and process browser evidence. Treat security
reports involving token exposure, secret capture, path traversal, report injection, contract parser
abuse, or CI privilege escalation as high priority.

Until a private security contact is published, do not post exploitable details in a public issue.
Open a minimal issue asking for a private reporting channel.

The current repository contains preparation material only and has no released runtime.
